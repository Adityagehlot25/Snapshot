# 🚀 Quick Deployment Reference

## Before You Start
- [ ] Backend code committed to GitHub
- [ ] Frontend code committed to GitHub  
- [ ] Both projects are in a monorepo or separate repos

---

## Step 1: Deploy Backend to Render (5 minutes)

```bash
# 1. Go to render.com
# 2. Click "New +" → "Web Service"
# 3. Connect GitHub repo
# 4. Fill in:
#    Name: ai-photo-backend
#    Environment: Python 3
#    Build Command: pip install -r requirements.txt
#    Start Command: uvicorn app.main:app --host 0.0.0.0 --port $PORT

# 5. Add Environment Variables:
#    ENVIRONMENT=production
#    FRONTEND_URL=[WILL_UPDATE_LATER]

# 6. Click "Create Web Service"
# 7. Wait for deployment (2-3 minutes)
# 8. Copy the URL: https://your-backend.onrender.com
```

---

## Step 2: Deploy Frontend to Vercel (5 minutes)

```bash
# 1. Go to vercel.com
# 2. Click "Add New..." → "Project"
# 3. Import GitHub repo (frontend folder)
# 4. Fill in:
#    Framework: Vite
#    Build Command: npm run build
#    Output Directory: dist

# 5. Add Environment Variable:
#    VITE_BACKEND_URL=https://your-backend.onrender.com

# 6. Click "Deploy"
# 7. Wait for deployment (1-2 minutes)
# 8. Copy the URL: https://your-frontend.vercel.app
```

---

## Step 3: Update Backend CORS (2 minutes)

```bash
# 1. Go to Render Dashboard
# 2. Select your backend service
# 3. Click "Environment"
# 4. Update FRONTEND_URL:
#    https://your-frontend.vercel.app
# 5. Click "Save"
# 6. Render will auto-redeploy (1 minute)
```

---

## Step 4: Test Everything (5 minutes)

```bash
# 1. Open frontend URL in browser
# 2. Upload an image
# 3. Verify:
#    ✓ Image appears in preview
#    ✓ Analysis loads
#    ✓ Sliders work
#    ✓ Download works
# 4. Check DevTools (F12) → Network tab
#    ✓ No CORS errors
#    ✓ API requests to backend URL
```

## Environment Variables Cheat Sheet

### Backend (.env on Render)
```
ENVIRONMENT=production
FRONTEND_URL=https://your-frontend.vercel.app
PORT=8000
```

### Frontend (.env on Vercel)
```
VITE_BACKEND_URL=https://your-backend.onrender.com
```

## If Something Goes Wrong

### CORS Error
```
Error: Access to XMLHttpRequest blocked by CORS

Fix: Check backend FRONTEND_URL environment variable
     Should be: https://your-frontend.vercel.app
                (exactly, no trailing slash)
```

### 404 on API
```
Error: POST http://localhost:8000/ 404

Fix: Check frontend VITE_BACKEND_URL
     Should be: https://your-backend.onrender.com
                (no trailing slash)
```

### Image Upload Fails
```
Error: Network error / 500

Fix: Check backend logs on Render
     Verify /uploads directory exists
     Check file permissions
```

## URLs Pattern

### Generated by Render
```
https://[project-name].onrender.com
```
Example: `https://ai-photo-backend.onrender.com`

### Generated by Vercel
```
https://[project-name].vercel.app
```
Example: `https://ai-photo-frontend.vercel.app`
